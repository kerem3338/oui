### The Syntax
OUI just has operations (functions) to work with, so function calls are seperated by a `---` and function arguments are seperated by a `\`
**Example:**
```text
print|'Hello World' --- normal exit
```

### The Current Operations
- `set-var`
- `do nothing for <x> amount of time`
- `del-var`
- `print`
- `speak to me`
- `...`
- `normal exit`
- `force exit`
- `force exit if you want`
- `normal exit if you want`
- `where`
- `print_var`
